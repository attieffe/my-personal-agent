# myJob — architettura documentale

Questa cartella raccoglie lavoro e documentazione in modo coerente (niente file “sparsi”).

## Struttura generale
- `COLZANI/*` → contenuti interni/tecnici per area
- `ATTILIO_A_CASA/*` → todolist personale “Attilio” da fare a casa (extra Colzani + personale)
- `DIRETTI/CLIENTI/*` → documentazione per cliente
- `GET_ME_DIGITAL/PROGETTI/*` → progetti
- `SINAPPS/PROGETTI/*` → progetti
- `EMAIL/*` → export/log email

## Convenzioni rapide
- Nomina cartelle in `snake_case` o `CapWords` (coerente con ciò che c’è già)
- Date in formato `YYYY-MM-DD`
- Ogni cartella “radice” ha un `00_index.md` con link ai contenuti

Vedi:
- `CONVENTIONS.md`
- `WORKFLOWS.md`
